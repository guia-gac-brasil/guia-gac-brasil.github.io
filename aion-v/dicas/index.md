# Alterar fuso horário

Passo a passo para alterar o fuso horário no Aion V através de ajuste de configurações no sistema do veículo:

1. Na tela inicial, clique no ícone do carro.
2. Acesse Definição do Sistema.
3. Clique em Sistema.
4. Em Autorização do Sistema do Veículo, selecione Definições.
5. Localize a opção Sistema de Desktop para acessar as opções.
6. Clique em Informações de Localização.
7. Desabilite a opção de Informações de Localização.
8. Clique em Sair.
9. Após sair, retorne para Definições Básicas dentro do menu Definição do Sistema.
10. Em Fuso Horário, altere para América/São_Paulo.
